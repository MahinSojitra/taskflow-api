Welcome to TaskFlow API A simple and efficient task management API TaskFlow API is a powerful and easy-to-use API designed to help you manage your tasks efficiently. It provides endpoints for user authentication, task creation, updates, and more.
Create an Account Start using TaskFlow API
POST Create Account
https://taskflowapi.vercel.app/api/users/signup
Create a new user account with name, email, and password. No authentication required.

API Documentation Check out the documentation for the API endpoints.

POST Sign In
https://taskflowapi.vercel.app/api/users/signin
Sign in with email and password to receive authentication tokens.
GET Get Profile
https://taskflowapi.vercel.app/api/users/profile
Get current user's profile information. Requires authentication.
PUT Update Profile
https://taskflowapi.vercel.app/api/users/profile
Update user profile information. Requires authentication.
POST Forgot Password
https://taskflowapi.vercel.app/api/users/forgot-password
Request a password reset link. Sends reset instructions to registered email.
POST Reset Password
https://taskflowapi.vercel.app/api/users/reset-password
Reset password using the token received via email.
POST Refresh Token
https://taskflowapi.vercel.app/api/users/refresh-token
Get a new access token using the refresh token. Requires authentication.
POST Sign Out
https://taskflowapi.vercel.app/api/users/signout
Sign out the current user and invalidate tokens. Requires authentication.
GET Get All Tasks
https://taskflowapi.vercel.app/api/tasks
Get all tasks for the authenticated user. Supports pagination and filtering.
POST Create Task
https://taskflowapi.vercel.app/api/tasks
Create a new task with title, description, priority, and due date. Requires authentication.
GET Get Task
https://taskflowapi.vercel.app/api/tasks/:id
Get details of a specific task by ID. Requires authentication.
PUT Update Task
https://taskflowapi.vercel.app/api/tasks/:id
Update task details including title, description, status, and priority. Requires authentication.
DELETE Delete Task
https://taskflowapi.vercel.app/api/tasks/:id
Permanently delete a task. This action cannot be undone. Requires authentication.
GET Get All Users
https://taskflowapi.vercel.app/api/users/all
Get a list of all registered users. Requires admin authentication.
GET Get All Users Tasks
https://taskflowapi.vercel.app/api/tasks/all
Get all tasks from all users. Requires admin authentication.
