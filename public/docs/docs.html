POST Sign In
https://taskflowapi.vercel.app/api/auth/signin
Sign in with email and password to receive authentication tokens.
GET Get Profile
https://taskflowapi.vercel.app/api/auth/profile
Get current user's profile information.
PUT Update Profile
https://taskflowapi.vercel.app/api/auth/profile
Update user profile information.
POST Forgot Password
https://taskflowapi.vercel.app/api/auth/forgot-password
Request a password reset.
POST Reset Password
https://taskflowapi.vercel.app/api/auth/reset-password
Reset password using the OTP received via email.
POST Refresh Token
https://taskflowapi.vercel.app/api/auth/refresh-token
Get a new access token using the refresh token.
GET Active Sessions
https://taskflowapi.vercel.app/api/auth/sessions
Get all active sessions for the current user.
POST Sign Out
https://taskflowapi.vercel.app/api/auth/signout
Sign out the current user session.
POST Sign Out All Devices
https://taskflowapi.vercel.app/api/auth/signout/all
Sign out from all devices including current device.
GET Get All Tasks
https://taskflowapi.vercel.app/api/tasks
Get all tasks for the authenticated user. Supports pagination and filtering.
POST Create Task
https://taskflowapi.vercel.app/api/tasks
Create a new task.
GET Get Task
https://taskflowapi.vercel.app/api/tasks/:id
Get details of a specific task by ID.
PUT Update Task
https://taskflowapi.vercel.app/api/tasks/:id
Update task details.
DELETE Delete Task
https://taskflowapi.vercel.app/api/tasks/:id
Permanently delete a task. This action cannot be undone.
GET Get All Users
https://taskflowapi.vercel.app/api/users/all
Get a list of all registered users.
GET Get All Users Tasks
https://taskflowapi.vercel.app/api/tasks/all
Get all tasks of all users.
